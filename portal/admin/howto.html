How To & Help

How to Upload a Development with Multiple Plots

  1. Go to the Developments section.
  2. Click Bulk Import.
  3. Download the sample CSV template and fill in your development and plot details.
    Each row is a plot. Use the same development name/location for all plots in the same site.
  4. Choose Add as New Development or select an existing development to add plots to.
  5. Upload your completed CSV and click Import.
  6. All plots will be created and visible in the Plots section.

How to Add a Plot

  1. Go to the Plots section.
  2. Click Add Plot.
  3. Fill in the plot details (number, development, type, bedrooms, price, status, etc.).
  4. Click Add Plot to save.
