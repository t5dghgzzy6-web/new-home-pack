Developer Help & How To

How to Upload a Development with Multiple New Home Plots

  1. Go to the Developments section in the Developer Portal.
  2. Click Bulk Import.
  3. Download the sample CSV template and fill in your development and new home plot details.
    Each row is a new home plot. Use the same development name/location for all new home plots in the same site.
  4. Upload the completed CSV file.
  5. Review the imported new home plots and confirm the details.

How to Add a New Home Plot

  1. Go to the New Home Plots section.
  2. Click Add New Home Plot.
  3. Fill in the new home plot details (number, development, type, bedrooms, price, status, etc.).
  4. Click Add New Home Plot to save.

How to Upload Documents to a Development

  1. Go to the Developments section in the Developer Portal.
  2. Select the development you want to update.
  3. Click Upload Documents.
  4. Choose the document from your computer or drag and drop it into the upload area.
  5. Add a document name and select the document type (if required).
  6. Click Save to finish.
  • The document is securely stored.
  • It becomes available to relevant users based on access settings.
  • You can replace or remove documents at any time.
Tips:
  • Use clear file names (e.g. Fire Safety Report – Phase 1).
  • Check you’re uploading to the correct development before saving.

Coming Soon: Assign Documents to Phases

You will be able to group new home plots into phases within a development. When you add a document (e.g., a search or report) to a phase, it will automatically be available to all new home plots in that phase—no need to upload to each new home plot individually.

Need Help?

  • Check on-screen help messages for guidance.
  • Contact platform support if something doesn’t look right.
  • If an upload fails, try refreshing the page and uploading again.
